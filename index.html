How we work

What stubborn good
actually means.

These aren't values we put on a wall. They're the standards we hold ourselves to when nobody's watching — and the reason we do this in the first place.

01
Grit

We don't quit when it gets hard. That's usually when it gets interesting. The word "stubborn" is in our name on purpose.

02
Innovation

We build things that didn't exist before. Not because it's trendy, but because the problem deserved a better answer.

03
Integrity

We do what we say. We build what we mean. We treat people — users, partners, everyone — the way we'd want to be treated.

04
Real impact

If it doesn't help real people live better lives, we're not interested. Metrics are nice. Meaning is the point.
